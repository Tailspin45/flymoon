ZIPCATCHER TRANSIT DETECTOR OVERVIEW

Zipcatcher's transit detector combines multiple specialized algorithms working in concert: The system begins with dual-signal detection—computing both consecutive-frame differences and disk-weighted reference differences with mean-subtraction for scintillation immunity—operating on a downsampled 160×90 canvas at ~15 fps. Hough circle detection locates the Sun/Moon disk every 2 seconds, ensuring signals are only computed within the inner disk (excluding a 25% limb margin to eliminate atmospheric shimmer false positives). PyWavelets performs level-3 sym4 discrete wavelet transform (DWT) detrending to isolate the 0.13–2s transit impulse while rejecting slow cloud-edge drift. Matched-filter templates (6–120 frames) with graduated hit-rate thresholds validate event duration, while an adaptive noise density guard monitors 60-second background windows to proportionally raise thresholds during high sunspot activity. Finally, detected events pass through a second-stage TransitCNN classifier—a compact ONNX-optimized convolutional neural network trained on real transit clips—which outputs binary logits to distinguish genuine aircraft transits from atmospheric artifacts, with post-capture frame-by-frame analysis using OpenCV blob detection, solar-disk masking, and cross-frame trajectory correlation to isolate and label the transit within the recording.
