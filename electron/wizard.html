Welcome to Zipcatcher

Zipcatcher detects aircraft transiting the Sun and Moon in real time, and can automatically trigger your Seestar telescope to capture the event.

  • Live flight tracking via 4 free ADS-B sources (OpenSky, ADSB-One, adsb.lol, adsb.fi)
  • Celestial position calculations (no internet needed for sky math)
  • Telegram push notifications for high-probability transits
  • Seestar telescope auto-capture control
  • Interactive map with transit overlay
FlightAware, Telegram and Seestar are all optional. You can skip straight through this wizard and Zipcatcher will run with free ADS-B sources only.

✈️ FlightAware API Key optional

Zipcatcher can optionally enrich high-probability transits with FlightAware metadata (aircraft type, registration, flight number). The free Personal tier includes 500 queries/month, which is plenty for typical use. Leave blank to run on free ADS-B sources only.

1. Sign up at flightaware.com/aeroapi/portal
2. Create a new API key under the "Personal" plan
3. Paste the key below

🗺️ Search Area

Zipcatcher searches for aircraft within a rectangular area around your location. A ±2° box (~220 km) works well for most observers — we've already filled this in for you based on your location. Click Next to accept, or tweak the corners below.

Please fill in all four bounding box values

📱 Telegram Notifications optional

Receive a push notification on your phone when a high-probability transit is detected. Skip this step if you only want the web interface.

1. On your mobile device, download the Telegram app and open a chat with @BotFather there
2. Send /newbot and follow the prompts to get a Bot Token
3. Send any message to your new bot from your phone, then open:
  api.telegram.org/bot{token}/getUpdates
4. Your Chat ID is the "id" field in the chat object

🔭 Seestar Telescope optional

If you have a Seestar on your Wi-Fi network in Station Mode, Zipcatcher can automatically start recording when a transit is imminent.

Make sure your Seestar is on the same Wi-Fi network and is in Solar or Lunar viewing mode before starting a monitoring session. See Help → Telescope Guide for detailed setup instructions.
You can enable Seestar later from File → Preferences.
